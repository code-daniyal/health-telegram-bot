\# Health Telegram Bot



This is a Telegram bot written in Python that helps users track their health.



\## Features



\* BMI calculation

\* Weight tracking

\* Health charts

\* Personal health data storage



\## Technologies



\* Python

\* aiogram

\* matplotlib

\* SQLite



\## How to run



1\. Install dependencies



pip install -r requirements.txt



2\. Add your Telegram token to config.py



3\. Run the bot



python main.py
